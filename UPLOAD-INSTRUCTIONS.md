# GitHub Pages upload instructions

1. Open the repository that publishes `https://guopengli-11.github.io/`.
2. Upload **the contents of this folder**, not the enclosing folder itself.
3. Keep the `assets` folder and its subfolders unchanged.
4. Replace the existing files when GitHub asks.
5. Commit the upload and wait one or two minutes for GitHub Pages to rebuild.
6. Open the website in a private browser window and refresh once.

The website no longer downloads fonts from Google. All required fonts, images,
styles and scripts are included locally in this package.
